# Snake Reimagined (React + Vite)

A modern Snake game built with React functional components and hooks.

## Tech Stack
- React + Vite
- Functional components
- Hooks: `useState`, `useEffect`, `useRef`, `useMemo`, `useCallback`

## File Structure

```text
.
├── index.html
├── package.json
├── vite.config.js
└── src
    ├── App.jsx
    ├── main.jsx
    ├── styles.css
    └── components
        ├── Controls.jsx
        ├── Food.jsx
        ├── GameBoard.jsx
        ├── ScorePanel.jsx
        └── Snake.jsx
```

## Setup

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```
